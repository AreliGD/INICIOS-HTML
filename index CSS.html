Cascading Style Sheets Hojas de Estilos en cascada


Existen 3 formas de agregar estilos a las paginas web:

  1. Interna:Desde head agregando la etiqueta style se coloca la condicionante como en un archivo de estilo de tipo ".css" con los atributos requeridos, un ejemplo de syntaxis en el comentario interno del codigo.
  2. Externa:Se llaman desde un documento externo de tipo .css que se encuentre dentro de la misma carpeta del archivo raíz, colocando las condicionantes de los atributos en formato css. de igual forma se realizan las llamdas de un segundo documento externo con el atributo @import url (direccion del documento) desde la primer pagina de css como una condicionante de estilo, sin embargo es una mala práctica.
  3. En línea: Se colocan directamente en la linea que se requiera aplicar el estilo y va con el atributo style dentro de la etiqueta principal del parrafo, ejemplo dentro del codico como comentario.
