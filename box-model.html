Modelo de Caja en CSS

    Modelo de Caja:

    Es la forma en la que css ve a los elementos de html, como sifueran cajas con las siguientes propiedades:

  1. contenido(content): El contenido (texto) del elemento html, tomando en cuenta sus dimensiones (width & height)
  2. borde:(border): Lo que delimita cada uno de elementos html
  3. relleno:(padding): Son las distancias internas (La distancia del borde al contenido)
  4. margen:(margin): Son las distancias externas (la distancia entre elementos html y sus elementos hermanos o padres)

    Dentro del modelo de caja hay que considerar que una caaja tendra 4 lados

  1. Arriba (top)
  2. Derecha (rigth)
  3. Abajo (bottom)
  4. Izquierda (left)
