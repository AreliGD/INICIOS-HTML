Etiquetas semanticas estructurales

Etiqueta contenedora que no tiene semantica
header - cabecera de un sitio web o de una sección
main - define la sección principal del documento, solo puede haber un main en el documento
footer-pie de pagina de un sitio web o de una sección
nav- representa una navegación o botonera
article-representa una sección de contenido(que por si sola se explica)
aside- representa un contenido complementario o secundario o publicidad
section-represanta una seccion de contenido generico
address-representa una dirección física o infromación de contacto

Etiquetas de Bloque VS Etiquetas de Linea

La etiqueta de bloque por excelenciqa es div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas
la etiqueta de linea por excelencia es span, una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido Hola soy otra linea span ☝️
